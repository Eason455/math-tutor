获取 API Key 指南

1 MiMo （拍照识别）
打开小米 MiMo 官网，找到「API Access」入口：
https://mimo.xiaomi.com
注册并登录后，进入「API Key 管理」页面
点击「创建 API Key」，复制生成的 Key
粘贴到上方 MiMo Key 输入框中
2 DeepSeek （题目分析）
打开 DeepSeek 开放平台：
https://platform.deepseek.com
注册并登录后，进入「API Keys」页面
点击「创建 API Key」，复制生成的 Key
粘贴到上方 DeepSeek Key 输入框中
提示：两个平台都需要注册，获取的 Key 填入对应输入框后点「保存」即可。Key 只存储在你的设备上，不会上传到任何服务器。
